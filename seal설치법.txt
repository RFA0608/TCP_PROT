1. wsl을 킨다
2. "seal"폴더를 만들고 해당 폴더로 이동한다
3. "git clone https://github.com/microsoft/SEAL.git"을 이용하여 SEAL라이브러리를 다운받는다.
4. "git clone https://github.com/Microsoft/vcpkg.git"을 이용하여 SEAL라이브러리의 패키지파일을 다운받는다.
5. vcpkg폴더로 이동하여 "./bootstrap-vcpkg.sh"를 실행한다.
5-1. 가끔 ninja가 안깔려있으면 에러가 뜨니 "sudo apt-get install -y ninja-build"를 설치
6. "./vcpkg integrate install"를 실행하고 "./vcpkg install seal"를 실행한다.
6-1. 가끔 clang이 안깔려있으면 에러가 뜨니 "sudo apt install clang"을 이용하여 설치한다.
6-2. pkg-config가 안깔려 있어도 에러 뜨니 "sudo apt-get install pkg-config"를 이용하여 설치한다.
6-3. 위에것을 다 설치했으면 다시 6번을 시행한다.
7. SEAL라이브러리로 이동하여 cmake를 이용한다. 명령어는 순서대로 "cmake -S . -B build", "cmake --build build", "sudo cmake --install build"이다.
7-1. cmake가 없으면 "sudo apt install cmake"를 이용하여 설치한다.
7-2. 위에것을 설치했으면 다시 7번을 시행한다.
8. 작업플레이스 폴더를 SEAL라이브러리 밖에 만든다.
9. 작업 플레이스에 "CMakeLists.txt"와 cpp파일을 만든다. 예제 코드는 해당 설치법 txt의 같은 폴더에 존재한다.
10. 작업 플레이스에서 "mkdir build"를 이용하여 build 폴더를 만든다.
11. build 폴더로 이동하여 "cmake .."을 이용하여 링커를 연결한다.
12. build 폴더에서 make를 이용하여 응용프로그램을 빌드한다.
13. 해당 폴더에 생긴 응용프로그램을 실행한다.
